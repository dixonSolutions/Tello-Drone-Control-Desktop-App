# Icon Placeholder

This directory should contain the application icons:

- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

You can generate these icons from a single source image using tools like:
- https://tauri.app/v1/guides/features/icons
- `npm run tauri icon path/to/icon.png`

For now, you can use placeholder icons or generate them from the Tauri CLI.
